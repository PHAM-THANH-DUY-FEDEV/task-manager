# 🚀 Task Manager SPA - ReactJS

Một ứng dụng quản lý công việc (Single Page Application) hiện đại, tập trung vào trải nghiệm người dùng mượt mà, quản lý tiến độ và lưu trữ nhật ký học tập.

---

## ✨ Tính năng chính

- **Quản lý Task toàn diện**: Thêm mới, xem chi tiết, và xóa công việc nhanh chóng.
- **Thống kê trực quan**: Thanh Progress Bar tự động cập nhật tiến độ theo trạng thái: _Vừa tạo, Đang làm, Hoàn thành_.
- **Tìm kiếm & Bộ lọc**: Hệ thống lọc theo trạng thái và tìm kiếm tiêu đề giúp tối ưu hóa quản lý.
- **Giao diện Responsive**: Tối ưu hóa hoàn toàn cho **Mobile** (layout cột) và **Desktop** (layout ngang).
- **Cảnh báo Deadline**: Tự động nhận diện các công việc sắp đến hạn hoặc quá hạn.
- **Nhật ký & Hiệu ứng**:
  - Viết nhật ký hàng ngày (CRUD & Search).
  - Ghi lại kỹ năng đã học kèm chức năng **Upload Ảnh & Video**.

## 🛠 Công nghệ sử dụng

- **Frontend**: ReactJS (Context API quản lý trạng thái).
- **Styling**: Tailwind CSS (Giao diện hiện đại, tinh tế).
- **Icons**: Lucide React.
- **Storage**: LocalStorage (Lưu dữ liệu ngay trên trình duyệt).
- **Server**: Node.js (Hỗ trợ upload file media).

## 🚀 Hướng dẫn cài đặt

Mở terminal và thực hiện các bước sau để chạy dự án trên máy cục bộ:

### 1. Clone dự án

```bash
git clone [https://github.com/PHAM-THANH-DUY-FEDEV/task-manager.git](https://github.com/PHAM-THANH-DUY-FEDEV/task-manager.git)
cd task-manager
```

### 2. Cài đặt thư viện và chạy dự án

```bash
npm install
npm run dev
node server.js
```
